# swing_multithread
Animație Java Swing (AWT) a unei mori de vânt cu arhitectură multi-thread: un thread rotește palele, unul detectează reflexia la ora 12, iar comunicarea între ele se face prin Exchanger&lt;Color>. Culoarea soarelui se propagă pe pale la fiecare trecere. Proiect demonstrativ pentru sincronizare între thread-uri în Java.
